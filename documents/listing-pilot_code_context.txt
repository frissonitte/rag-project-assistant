dolap_core.py
~~~~~~~~~~~~~
Core automation engine for Dolap marketplace.

Manages an Appium-controlled Android session and exposes a single entry point
`run_marketplace_bot` that iterates over the seller's product list and applies
the configured operation (price drop, price fix, price increment, or listing
refresh / description update).

All platform-specific selectors and Appium capabilities are loaded from
``automation_targets.private.json`` via :mod:`target_profiles`, so the core
logic stays platform-agnostic.

--- Function: def run_marketplace_bot(cfg: MarketplaceBotConfig) -> None ---
"""Run the automation loop according to *cfg*.

The loop:
1. Opens the product list.
2. Iterates over visible products, processing each unseen one.
3. Scrolls forward when all visible products have been processed.
4. Stops after ``cfg.max_scrolls_without_new_items`` consecutive scrolls
   that yield no new products.

Depending on which flags are set in *cfg*, each product is either:
- **Price-drop / fix / increment mode** (``enable_price_drop=True``):
  taps the inline price element and updates it directly from the list view.
- **Listing refresh mode** (default): enters the listing detail, optionally
  updates the description, then re-submits the listing for approval — which
  refreshes its position in search results."""

gardrops_price.py
~~~~~~~~~~~~~~~~~
Standalone price-update bot for the Gardrops Android marketplace app.

Unlike the Dolap bot (which loads selector config from an external JSON file),
this script hard-codes Gardrops-specific resource IDs directly.  This is
intentional: Gardrops has fewer screens to automate and the selectors rarely
change.

Modes (set ``MODE`` constant below):
  ``"map"``        — look up each price in ``PRICE_MAP`` and apply the mapped value.
  ``"drop"``       — subtract ``PRICE_DROP_AMOUNT`` from every price.
  ``"increment"``  — add ``PRICE_INCREMENT_AMOUNT`` to every price
                     (capped at 249 when starting below 249).

Usage::

    python gardrops_price.py

CONSTANT: MODE = 'map'

CONSTANT: PRICE_MAP: dict[int, int] = {67: 82, 100: 115, 135: 150, 167: 182, 170: 185, 192: 207, 200: 215, 230: 250}

CONSTANT: PRICE_DROP_AMOUNT = 1

CONSTANT: PRICE_INCREMENT_AMOUNT = 1

CONSTANT: MAX_SCROLLS_WITHOUT_NEW_ITEMS = 10

CONSTANT: INCREMENT_CAP_FROM_BELOW_249 = 249

CONSTANT: MIN_ALLOWED_PRICE = 0

target_profiles.py
~~~~~~~~~~~~~~~~~~
Loads automation target profiles from ``automation_targets.private.json``.

The private JSON file is **not** committed to version control (it is listed in
``.gitignore``).  Copy ``automation_targets.example.json`` to
``automation_targets.private.json`` and fill in your own selectors before
running any bot script.
